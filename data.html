Format data for analysis

Download the practice sheet: tutorial_spreadsheet.xlsx

Use clear column names and one meaning per column; avoid mixed entry styles (e.g., “Y/Yes/1”). Keep a raw copy and clean a working copy.

Create dichotomous variables

Why 0/1?

Binary (0/1) columns make filtering and group comparisons easy later in Excel or SPSS.

Convert categories to separate 0/1 columns (e.g., tutorial_type_video, has_experience), keeping the original category column.

Run descriptive statistics

Excel → File > Options > Add-ins → enable Analysis ToolPak. Then Data > Data Analysis > Descriptive Statistics on continuous variables; output to a new sheet.

Run inferential statistics

Choose a test that fits your question and data:

  • Two groups: t-test or Mann–Whitney U
  • 3+ groups: ANOVA or Kruskal–Wallis
  • Association: Pearson or Spearman correlation

Example: compare usefulness by tutorial type with Kruskal–Wallis: Analyze → Nonparametric Tests → K Independent Samples.
